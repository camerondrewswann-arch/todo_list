# To-Do List CLI Application

## Overview
This is a simple command-line To-Do List application built in Python. It allows users to manage tasks by adding, viewing, and deleting them.

## Features
- Add tasks
- View tasks
- Delete tasks
- Input validation and error handling

## How to Run
1. Make sure Python is installed
2. Download or clone this repository
3. Run the program:

```bash
python todo_app.py# todo_list
